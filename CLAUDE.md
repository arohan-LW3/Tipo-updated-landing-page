# TIPO Project

## Overview
TIPO is a product landing page designed to be accessed via QR codes on bottles. The page serves as a link-in-bio style hub for the TIPO brand.

## Brand
- **Name**: TIPO
- **Logo**: `assets/tipo_logo_LOGO.svg`
- **Brand Colors**: Gold/Yellow (#FDCA00), Red (#E60004), Grey (#68676C)
- **Logo includes**: A chili pepper motif between the "I" dots

## Project Structure
- `/assets/` — Brand assets (logo SVG)
- `/landing/` — Landing page (QR code destination)
- `/index.html` — Root index

## Landing Page (`landing/index.html`)
- **Purpose**: Product QR code landing page attached to bottles
- **Style**: Linktree-inspired, mobile-first, single-column vertical layout
- **Design direction**: Clean, minimal link hub with brand identity, profile/logo area at top, stacked link buttons below
- **Key design cues from inspiration**:
  - Centered vertical layout
  - Rounded buttons with subtle shadow
  - Warm color palette matching brand colors
  - Mobile-optimized
  - Social media icon row
  - Clean typography

## Tech Stack
- Static HTML/CSS/JS (no build tools)
- Single-page, self-contained
